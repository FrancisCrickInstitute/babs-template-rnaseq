## Introduction

This site provides the documentation on the BABS bulk RNASeq
pipeline. There are two very distinct aspects to the documentation,
and they'll both be covered on this site:

- How the github repository can turn itself into the starting point
  for a new project (e.g. obviously we don't want all the meta
  information, like this page, ending up in a user's project)
  
- What to do once you've got such a starting point for a new project.
